# File Rift

## A tool for converting protobuf files in Swordigo to plain text.

## Installation:

Just donwload, unpack and run.

## Usage:

Run `fileRift3.6.py` using python.

e.g.

```python
$ python3 fileRift3.6.py
```

To change the configuration, edit `riftOptions.py`.
